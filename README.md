# Landing Page UI

Live Demo:
https://cool-marshmallow-251ea1.netlify.app

Frontend landing page prototype.

This project demonstrates a simple landing page interface built with basic web technologies.

## Features

- modern landing page layout
- responsive UI structure
- service price submission interface (UI only)

## Important

This is only a frontend prototype.

The form does not send data anywhere and is used only to demonstrate the interface.

## Technologies

HTML  
CSS  
Vanilla JavaScript
